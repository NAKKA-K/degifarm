## 機能
団体ごとのグループを作成して、電子媒体のアップロードやダウンロードを提供。
アカウント管理によって権限機能を実装。

## 対象
一般的な学校、専門学校を対象としている

---

## パッケージのインストール

    $ pip install -r requirements.txt

## テストサーバーの起動

    ./runserver.sh
