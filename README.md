## 環境設定
### Git
**Windows**
下記のURLからインストーラをダウンロードして、手順通りに進めていけばインストールできます。
ULR:<https://git-for-windows.github.io/>
※特にこだわりが無ければ、初期設定のまま先に進めれば問題ありません。

**Mac**
今回は割愛

**Linux**
Debian系統のディストリビュージョンであれば
    $ apt-get install git

Fedora系統のディストリビュージョンであれば
    $ yum install git

でインストールできます。
